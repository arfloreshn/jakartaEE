Componente Panel Menu



Objetivos:

• Aprendar los conceptos basicos de como crear menus, submenus e items de una aplicación web con Panel menu.
• Aprendar los conceptos basicos de comunición con bases de datos relacionales usuando JPA y procedimientos almacenados.
• Aprendar los conceptos basicos del uso de plantillas y contenedores con Java Server Faces.
• Aprendar a desplegar aplicaciones web Java EE con Wildfly Server y Payara Server.

A que publico me dirijo:
A programadores TRANSACCIONALES, cuyo consumidor final seran usuarios internos de aplicaciones web.
no son para marketing digital ó leading pages o modulos CRM bootcam o chatRoom.

Consideraciones:
Estas lecciones son para programadores que estaran trabajando con usuario internos de aplicaciones web Java EE modulares o monoliticas

Principiante
